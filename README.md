# A Reproducible Comparison of Classical and Deep Learning Models for High-Frequency NILM Using Harmonic Features

This repository provides the full experimental framework, code, and results
associated with the paper:

**A Reproducible Comparison of Classical and Deep Learning Models for
High-Frequency NILM Using Harmonic Features**

submitted to the SBRC conference.

## ðŸ“Œ Overview

Non-Intrusive Load Monitoring (NILM) aims to identify individual appliance
consumption from aggregate electrical measurements. This work evaluates the
performance of classical machine learning and deep learning models on a
high-frequency NILM dataset based on harmonic features.

The repository emphasizes **reproducibility**, providing a unified benchmark
covering nine learning architectures.
